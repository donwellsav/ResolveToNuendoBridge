# Resolve to Nuendo Conform Bridge

Desktop-first internal translator app scaffold for Resolve -> Nuendo conform operations.

## Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-style component structure

## Run
```bash
npm install
npm run dev
```

## Phase 1 Notes
- Frontend-only
- Domain and UI backed by mock data
- Import/export services are stubs only
- localStorage persistence intentionally deferred
